Definitions

Pop-Up Box

Total Number of Ports - The number of Phase Enabled ports + Non-Phase Enabled ports at a site.

Phase Enabled Ports - Phase Enabled ports are used by Mobile Backhaul for 5G. Mobile Networks require phase synchronisation to ensure both packets arrive in the correct order for services such as voice and video as well as hand over between cell sites and accurate communication with the network.

In-Build Ports - The number of ports that currently have jobs that are currently scheduled and in progress.

Mobile Backhaul - All Core Metro Sites and All 10G Tier 1 Sites

All Ethernet - All Core Metro Sites, All 10G Tier 1 Sites, and All 10G 3rd Party.

Phase Enabled - All sites that are capable of selling Phase Enabled services

 

Available Capacity

4+ - Sites which have 4 or more 10G ports (Green)

1-3 - Sites which have between 1-3 10G ports (Amber)

0 - Sites which have 0 10G ports (Red)
