The Report provide per site capacity shortfall summary for 7750 SR-12 Device with user selected product type of Ethernet giving a count of Total number of ports, Used ports, Reserved ports, Free ports, cabled, Delta, Free VU spaces of Adva Rack and forecast in read only format.

Site: Name of the Network site

1141 Code: 1141 code of the Network site

Total Available Capacity: Total number Free and Reserved Ports

Reserved Ports: Total number of reserved ports per site which is reserved for selected product type.

Free Ports: Total number of free ports per site which can be used for Planning. Meaning it should not count any faulty ports.

Sub Category for 10G Port Speed – 10G HE (All Ports with Ethernet Access flag), 10G WMC – Total free ports on card with diversity + Ethernet Access Flag.

Product Line Forecast: It should show total count of port based on the product line forecast data upload by accounting user and the filter selection on port speed.

Port Build so far: It should show the count of total ports which has inventory status as “In-service” and should be free/reserved for current financial year

Ports Cabled: Total Ports connected to a patch panel (IBW) and is free

Free Rack Spaces: Total free VU rack spaces.

Sub-category: EAD,OSA,OSA-FC, Small Cells.

EAD: Ethernet Access Direct offers permanently connected, point-to-point high speed data circuits that provide a secure and un-contended access service for Communications Providers.

OSA: Optical Spectrum Access, A range of Openreach Optical DWDM (Dense Wavelength Division Multiplexing) products provided to Communication Providers.

OSA-FC: Optical Spectrum Access-Filter Connect.

Rack VU free space Calculation

  • Find out free ports on 7750
  • Find out if there is a PhysicalConnection/Conductor on it with specname as ‘IBW’
  • Traverse and find out the PP where it is connected and get the 1141 code of the RACK where that PP is located
  • Match the Rack1141 code of PP with the Rack 1141 code from Bert sheet
  • If matched, check the number of free Vu spaces on the those rack and display on the report

Ports To-be Cabled: It should display total number of ports = Total available capacity – Ports cabled
